# Fullcount.kr 디자인 시스템

## 기존 사이트 스타일 분석
- **배경:** 연한 베이지/미색 (#F5F5F0 계열)
- **카드:** 흰색 배경, 부드러운 그림자
- **텍스트:** 진한 차콜/블랙
- **하단 네비게이션:** 흰색 배경, 아이콘+텍스트
- **전체 톤:** 깔끔하고 미니멀, 정보 중심
- **로고:** "fullcount.kr" 텍스트 로고, 볼드 산세리프

## 디자인 원칙
1. 기존 fullcount.kr의 미니멀하고 깔끔한 톤 유지
2. 친근하고 접근하기 편한 느낌 (둥근 모서리, 부드러운 그림자, 여백)
3. 팀 구분은 대표 색상 3가지로 표현 (캐릭터/그림 없이)
4. 모바일 퍼스트, 데스크톱에서는 하단 탭바 숨기고 상단 메뉴 사용
